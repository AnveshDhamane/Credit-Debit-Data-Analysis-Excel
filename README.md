# Bank Debit & Credit Analysis Dashboard


## Project Overview

The dashboard is designed to help analyze banking operations across multiple banks and branches.
It highlights key performance indicators (KPIs), monthly trends, purpose-wise transactions, and branch-level financial insights.

The project demonstrates skills in:
- Excel Data Cleaning
- PivotTable & PivotChart Development
- KPI Cards
- Slicers & Filters
- Data Visualization
- Dashboard Formatting

## Features

### 1. Key KPIs

## Dataset Description

The dataset contains customer transaction records across multiple banks.
Below is the data dictionary explaining each column.

## Data Dictionary

| Column Name | Description |
|-------------|-------------|
| Customer ID | Unique identifier assigned to each customer. |
| Customer Name | Full name of the customer performing the transaction. |
| Account Number | Bank account number associated with the transaction. |
| Transaction Date | The date on which the transaction was performed. |
| Transaction Type | Indicates whether the transaction is Credit or Debit. |
| Amount | Monetary value of the transaction. |
| Balance | Account balance after the transaction was completed. |
| Description | Purpose of the transaction (Utility bills, Shopping, Salary deposit, etc.). |
| Branch | Bank branch where the transaction occurred. |
| Transaction Method | Mode of transaction (UPI, ATM, Cheque, Cash, Online Banking). |
| Currency | Currency used for the transaction (e.g., INR). |
| Bank Name | Name of the bank involved in the transaction. |


## Key Insights

### 1. Balanced Credit and Debit Flow
Credit and debit values are nearly equal, indicating a stable transaction environment.

### 2. High Activity in Major Banks
HDFC, ICICI, and Axis Bank have the highest transaction volumes.

### 3. Branch Performance Varies
Main and East branches show significantly higher balances.

### 4. Transaction Peaks by Month
Certain months show higher activity, likely due to seasonal or salary-cycle patterns.

### 5. Consumer Transactions Dominate
Utility bills, groceries, and online shopping form the majority of transactions.

### 6. Digital Payments Lead
UPI and online banking are the most commonly used methods.

## Recommendations

### 1. Promote High-Performing Branches
Use strong branches for targeted marketing and premium product promotions.

### 2. Improve Low-Performing Branches
Launch localized offers and improve services in lower-performing regions.

### 3. Enhance Digital Banking Experience
Add rewards and improve the user experience for digital channels.

### 4. Identify High-Value Customers
Frequent credit customers can be targeted for loan or investment products.

### 5. Optimize for High-Activity Months
Plan staffing and digital capacity around peak periods.

### 6. Strengthen Customer Retention
Banks with negative net flow should offer better incentives and service quality.
